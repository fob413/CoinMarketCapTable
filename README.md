# CoinMarketCap

## Features
* React application displays a table of the top 10 results from a crytocurrency API,
  [CoinMarketCap](https://coinmarketcap.com/api/documentation/v1/#)
* Implements pagination
* Refetches every 2 minutes

## Technologies Used
* Typescript (React)
* React Query
* Mantine for UI
* Jest for testing
